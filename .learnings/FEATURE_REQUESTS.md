# matlab-agent 用户需求记录

> 本文件记录用户提出的功能需求、改进建议和体验反馈。
> 由 matlab-agent 自我改进机制（Layer 1: 被动学习）自动维护。
>
> 触发关键词：
> - "能不能/可不可以/有没有办法..." → feature_request
> - "要是能...就好了" → feature_request
> - "这个功能不好用" → improvement

---

## 记录模板

```markdown
## [FEAT-YYYYMMDD-XXX] capability_name

**Logged**: ISO-8601 timestamp
**Priority**: medium
**Status**: pending | planned | implemented | rejected

### Requested Capability
用户想要什么能力

### User Context
为什么需要，解决什么问题

### Complexity Estimate
simple | medium | complex

### Suggested Implementation
如何实现

### Metadata
- Frequency: first_time | recurring
- Related Features: existing_feature_name
```
